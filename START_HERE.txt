🔧 HOW IT WORKS:

   BUILDING THE GAME:
   1. You double-click run.bat
   2. Script checks if Java is installed
   3. Script compiles all Java files → *.class files
   4. Script creates bin/ folder structure
   5. Script generates MANIFEST.MF with Main-Class: Main
   6. Script builds jar → EndlessRunner.jar
   7. Game is ready to play!

   PLAYING THE GAME:
   1. You double-click play.bat or EndlessRunner.jar
   2. Java executes the JAR file
   3. Main class is loaded
   4. GamePanel starts
   5. Game window appears
   6. Start playing!

   IMAGE LOADING:
   1. On startup, ImageLoader tries to load PNG files
   2. First checks: resources/player.png
   3. Then checks: resources/obstacle.png
   4. Then checks: resources/background.png
   5. If found: displays custom image
   6. If not found: displays fallback color/shape
