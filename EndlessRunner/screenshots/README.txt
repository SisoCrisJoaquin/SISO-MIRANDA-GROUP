SCREENSHOTS FOLDER

Required screenshot files:
1. menu-screen.png       - Main menu screen showing "Motorcyclist Seeker" title
2. gameplay-screen.png   - In-game action screenshot with player and obstacles

Place PNG images in this folder with the exact filenames above so they display in the README.md file.
