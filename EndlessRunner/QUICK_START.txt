🎯 TO PLAY THE GAME - 2 STEPS:

  STEP 1: Click run.bat
     → This builds your game into EndlessRunner.jar
     → Compiles all Java files automatically
     
  STEP 2: Click play.bat (or double-click EndlessRunner.jar)
     → The game starts immediately!

🖼️ TO ADD CUSTOM IMAGES:

  1. Create or find 3 PNG images:
     • player.png (30×40 pixels) - Your player character
     • obstacle.png (40×50 pixels) - What to avoid
     • background.png (800×500+ pixels) - The environment
     
  2. Place them in the resources/ folder
  
  3. Click run.bat to rebuild
  
  4. Click play.bat to see your custom graphics!

🆘 TROUBLESHOOTING:

  "run.bat says Java is not installed"
  → Download Java from oracle.com/java, restart, try again
  
  "Game won't build"
  → Make sure all .java files are in src/ folder
  → Try deleting the bin/ folder and running run.bat again
  
  "Images not showing"
  → Create resources/ folder if it doesn't exist
  → Make sure image files are PNG format
  → Use exact names: player.png, obstacle.png, background.png
  → Run run.bat again to rebuild

  For more help → Read SETUP.md
