IMAGE LOADING PROCESS:
══════════════════════

ImageLoader.loadImage("player.png")
           ↓
Check image cache
├─ If found → return cached image
└─ If not found → continue
           ↓
Try loading from JAR resources
├─ Success → cache and return
└─ Not found → continue
           ↓
Try loading from file system
├─ Check: resources/player.png exists?
├─ Yes → load, cache, return
└─ No → continue
           ↓
Try alternate path
├─ Check: resources/player.png exists?
├─ Yes → load, cache, return
└─ No → continue
           ↓
⚠️  Image not found
├─ Print error message
└─ Return null (use fallback graphics)


GAME RENDERING WITH IMAGES:
═════════════════════════════

If playerImage != null AND useImage == true:
    g2d.drawImage(playerImage, x, y, width, height, null)
Else:
    Draw fallback red rectangle
    
    
If obstacleImage != null AND useImage == true:
    g2d.drawImage(obstacleImage, x, y, width, height, null)
Else:
    Draw fallback rock shape
    
    
If backgroundImage != null AND useBackgroundImage == true:
    Draw background with parallax scrolling
Else:
    Draw fallback colored scene

FULL BUILD COMMAND SEQUENCE (What run.bat does):
═════════════════════════════════════════════════

@echo off
REM Check Java exists
java -version

REM Create directories  
mkdir bin
mkdir resources (if needed)

REM Compile Java
cd src
javac -d ../bin Main.java
javac -d ../bin GamePanel.java
javac -d ../bin Player.java
javac -d ../bin Obstacle.java
javac -d ../bin Scoreboard.java
javac -d ../bin ImageLoader.java
cd ..

REM Create META-INF
mkdir bin\META-INF

REM Create MANIFEST.MF
echo Manifest-Version: 1.0 > bin\META-INF\MANIFEST.MF
echo Main-Class: Main >> bin\META-INF\MANIFEST.MF

REM Build JAR
cd bin
jar cfm ../EndlessRunner.jar META-INF\MANIFEST.MF *.class
cd ..

RESULT: ✓ EndlessRunner.jar created!
